# Creating a new plan directly from a saved segment

*Insert Edit Segment screen shot...*  

**IMPORTANT:**  
Do not attempt to create a new plan directly from an unsaved new or edited segment. All segments must be saved first.  

To create a new plan directly from an existing segment, do the following:  

1. Click the **Segments** tab.  
  The **My segments** page displays.  
2. Locate the required segment using the **Segments list** or **Segments cards** view.  
3. Click the required segment.  
  The **Edit segment** page displays.  
1. Click **Start new plan**.  
  The **New plan** page displays.  
1. Follow the instructions given under [Creating new plans](CreatingNewPlans).
